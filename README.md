# EarthNow 🌍

A futuristic real-time live counter web app that visualizes global activity happening right now.

Users can explore live counters for:

* Internet activity
* Social media engagement
* Human behavior
* Environment statistics
* Technology usage
* Space activity
* Gaming
* Economy
* Sports
* And much more

Built with:

* HTML
* Tailwind CSS
* Vanilla JavaScript
* Lucide Icons

---

# ✨ Features

## 🌐 Real-Time Live Counters

Counters continuously increase using simulated real-time updates.

Examples:

* 📧 Emails Sent
* ❤️ Instagram Likes
* 🌫️ CO₂ Emitted
* 🎮 Active Gamers
* 🔍 Google Searches
* ⚡ Electricity Usage
* 🛰️ Satellites in Orbit

---

## ⚔️ Live Counter Battles

Dynamic hero section comparing two live counters.

Examples:

* 📧 Emails vs ❤️ Likes
* 🌱 Trees Planted vs 🪓 Trees Cut
* ⚡ Energy Usage vs 🌫️ CO₂

Features:

* Auto rotating battles
* Swipe support on mobile
* Manual navigation
* Animated transitions

---

## 📱 Mobile First Design

Optimized for:

* Mobile phones
* Tablets
* Desktop screens

Includes:

* Responsive typography
* Touch gestures
* Smooth scrolling
* Bottom navigation

---

## ❤️ Favorites System

Users can:

* Save favorite counters
* View favorite-only dashboard
* Remove favorites
* Clear all favorites

Favorites are stored using:

```js
localStorage
```

---

# 🖥️ Fullscreen Live Dashboard

Each counter opens into a fullscreen analytics view.

Includes:

* Massive live number display
* Live trend chart
* Real-time rate indicator
* Meaning/impact card
* Related counters
* Fun facts

Example:

```txt
🌫️ CO₂ Emitted Today

110,482,224

LIVE

[Trend Chart]

+1200/sec

[Impact Card]

[Related Counters]

[Fun Fact]
```
